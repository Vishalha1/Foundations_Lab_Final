## Theory Questions

### 1. Why should an operator run a ping sweep before doing deeper port scans?

A ping sweep helps identify which hosts are alive before running deeper scans. This saves time because the operator does not waste effort scanning inactive addresses. It also helps build a basic map of the network before moving into service and version detection.

### 2. Why is service version detection important during network enumeration?

Service version detection is important because open ports alone do not show the full risk. Knowing the exact service and version helps security teams identify outdated software, misconfigurations, and possible vulnerabilities that may affect a specific host.
